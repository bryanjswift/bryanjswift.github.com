Legacy Flash Players

In order to get around this, it is necessary to delete the registry entry. It is available at HKEY_LOCAL_MACHINE\SOFTWARE\Macromedia\FlashPlayer\SafeVersions.

To do so, follow these steps.

  1. Go to the start menu and select RUN
  2. Type cmd to open the command prompt
  3. Once in the command prompt, type regedit, which will open the registry
  4. Open HKEY_LOCAL_MACHINE
  5. Open Software
  6. Open Macromedia
  7. Open Flash player
  8. Select SafeVersions
  9. Select the appropriate plug-in version entry and delete it.
  10. Run the Flash installer and verify the version on the adobe website
